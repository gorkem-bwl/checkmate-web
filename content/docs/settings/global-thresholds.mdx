## Overview

Global thresholds set default alert levels for infrastructure monitors. These thresholds determine when alerts are triggered for resource usage.

## Configuring thresholds

1. Navigate to **Settings** in the sidebar
2. Scroll to the **Global thresholds** section
3. Set threshold percentages for CPU, memory, disk, and temperature
4. Click **Save** at the bottom of the page

- **CPU threshold**: Alert when CPU usage exceeds this percentage (1-100%)
- **Memory threshold**: Alert when memory usage exceeds this percentage (1-100%)
- **Disk threshold**: Alert when disk usage exceeds this percentage (1-100%)
- **Temperature threshold**: Alert when temperature exceeds this value (1-150°C)
