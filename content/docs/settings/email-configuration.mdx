## Overview

Configure SMTP settings to enable email notifications. This is required for email alerts and team invitations.

## SMTP configuration

1. Navigate to **Settings** in the sidebar
2. Scroll to the **Email** section
3. Enter SMTP server hostname and port
4. Enter authentication credentials (username and password)
5. Set the from email address
6. Configure TLS/SSL options as needed
7. Click **Send test email** to verify, then **Save**

<Callout type="info">
Only Admins and Superadmins can configure SMTP settings.
</Callout>
