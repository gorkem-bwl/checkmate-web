## Overview

The Checkmate dashboard gives you a complete view of your monitoring status. This article explains the main navigation areas and key metrics.

## Sidebar navigation

The left sidebar provides access to all major features:

- **Uptime**: View and manage HTTP, ping, port, and Docker monitors
- **PageSpeed**: Access performance monitoring and Lighthouse reports
- **Infrastructure**: Monitor server hardware metrics
- **Notifications**: Configure alert channels (email, Slack, Discord, etc.)
- **Checks**: View recent check results across all monitors
- **Incidents**: View current and past downtime events
- **Status pages**: Create and manage public status pages
- **Maintenance**: Schedule planned maintenance windows
- **Logs**: View system and application logs
- **Settings**: Manage account and global settings

## Monitors list

The main monitors view shows all your monitors with key information at a glance:

- **Status indicator**: Green (up), red (down), or yellow (degraded)
- **Name**: The monitor's display name
- **URL/Host**: The target being monitored
- **Response time**: Latest response time in milliseconds
- **Uptime**: Percentage uptime over the selected period

## Filtering and sorting

You can filter monitors by status (up, down, paused) and sort by name, status, or response time. Use the search box to quickly find specific monitors.
