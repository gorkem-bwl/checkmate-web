## Overview

Checkmate is an open-source uptime and infrastructure monitoring platform. It helps you track website availability, server performance, and service health—all from a single dashboard.

Whether you're monitoring a personal project or managing enterprise infrastructure, Checkmate provides the tools you need to stay informed about your systems.

## Monitoring types

- **HTTP/HTTPS**: Check website availability and response validation
- **Ping**: Verify server reachability using ICMP
- **TCP port**: Monitor specific services like databases and mail servers
- **Docker**: Monitor Docker container status and health
- **PageSpeed**: Track Core Web Vitals and performance scores
- **Infrastructure**: Monitor hardware metrics with the Capture agent
