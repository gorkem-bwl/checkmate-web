## Overview

Checkmate has three user roles with different permission levels. Assign roles based on what each team member needs to do.

## Role comparison

| Permission | User | Admin | Superadmin |
| --- | --- | --- | --- |
| View monitors | ✓ | ✓ | ✓ |
| Create/edit monitors | — | ✓ | ✓ |
| Manage notifications | — | ✓ | ✓ |
| Invite team members | — | ✓ | ✓ |
| Configure SMTP | — | — | ✓ |
| Manage user accounts | — | — | ✓ |
