## Overview

Matrix integration sends alerts to Matrix rooms using the Matrix protocol. Ideal for teams using decentralized messaging.

## Setting up Matrix

1. Get your Matrix homeserver URL
2. Create an access token for your bot account
3. Get the room ID where alerts should be sent
4. In Checkmate, navigate to **Integrations**
5. Create a new **Matrix** integration
6. Enter the homeserver, token, and room ID
7. Click **Test** then **Save**
