## Overview

Email notifications send alerts directly to your inbox when monitors detect issues. Configure SMTP settings to enable email delivery.

## Setting up SMTP

1. Navigate to **Settings** in the sidebar
2. Select the **Email** tab
3. Enter your SMTP server details
4. Configure authentication (username/password)
5. Set the sender email address
6. Click **Save** and **Test** to verify

## SMTP settings

- **Host**: SMTP server address (e.g., smtp.gmail.com)
- **Port**: SMTP port (typically 587 for TLS, 465 for SSL)
- **Username**: Authentication username
- **Password**: Authentication password or app password
- **From address**: Email address that alerts come from

<Callout type="info">
For Gmail, use an App Password instead of your regular password. Enable 2FA and generate an app password in your Google Account settings.
</Callout>
