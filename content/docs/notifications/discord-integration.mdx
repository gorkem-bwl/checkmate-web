## Overview

Discord integration sends alerts to your Discord server using webhooks. Ideal for development teams already using Discord for communication.

## Creating a Discord webhook

1. Open Discord and go to your server
2. Click the gear icon next to the target channel
3. Select **Integrations** → **Webhooks**
4. Click **New Webhook**
5. Name the webhook and copy the URL

## Configuring in Checkmate

1. Navigate to **Integrations** in the sidebar
2. Click **Create integration**
3. Select **Discord**
4. Paste your webhook URL
5. Click **Test** then **Save**
