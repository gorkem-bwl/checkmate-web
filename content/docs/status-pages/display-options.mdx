## Overview

Configure what information visitors see on your status page. Balance transparency with the level of detail you want to share.

## Display options

- **Show uptime percentage**: Display overall uptime statistics for each monitor
- **Show charts**: Display response time performance graphs
- **Show admin login link**: Add a link to the Checkmate login page
- **Publish/Unpublish**: Control whether the page is publicly accessible

<Callout type="info">
Unpublished status pages are only accessible by team members logged into Checkmate.
</Callout>
