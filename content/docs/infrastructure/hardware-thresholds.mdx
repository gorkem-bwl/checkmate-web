## Overview

Hardware thresholds define when Checkmate sends alerts about resource usage. Set thresholds to catch problems before they affect your services.

## Configuring thresholds

1. Navigate to **Infrastructure** in the sidebar
2. Click on a server to view details
3. Click **Settings** or the gear icon
4. Set threshold percentages for CPU, memory, and disk
5. Click **Save**

## Recommended thresholds

| Resource | Warning | Critical |
| --- | --- | --- |
| CPU | 70% | 90% |
| Memory | 80% | 95% |
| Disk | 80% | 90% |

<Callout type="info">
Start with these defaults and adjust based on your application's normal behavior.
</Callout>
