## Overview

Capture is the companion agent for infrastructure monitoring. Install it on your servers to collect CPU, memory, disk, and network metrics.

## Requirements

- Linux, macOS, or Windows server
- Network access to your Checkmate instance
- Minimal resources (< 50MB RAM, < 1% CPU)

## Installation

1. Navigate to **Infrastructure** in the sidebar
2. Click **Add server**
3. Copy the installation command
4. Run the command on your server
5. The server appears automatically once connected

<Callout type="info">
Capture is open source. View the source code at github.com/bluewave-labs/capture
</Callout>
