## Overview

Disk monitoring tracks storage space and I/O performance. Running out of disk space can cause application failures and data loss.

## Disk metrics

- **Total space**: Total disk capacity
- **Used space**: Space currently in use
- **Available space**: Free space remaining
- **Usage percentage**: Percentage of disk used

<Callout type="warn">
Set alerts at 80% disk usage to give yourself time to add storage or clean up files before running out of space.
</Callout>
