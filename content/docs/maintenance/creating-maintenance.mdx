## Overview

Maintenance windows suppress alerts during planned downtime. Schedule them before deployments or infrastructure work to avoid false alarms.

## Creating a maintenance window

1. Navigate to **Maintenance** in the sidebar
2. Click **Create maintenance window**
3. Enter a name describing the maintenance
4. Select the affected monitors
5. Set the start and end times
6. Click **Create**

<Callout type="info">
Monitors continue running during maintenance, but incidents won't trigger notifications.
</Callout>
