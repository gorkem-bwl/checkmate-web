## Overview

Manage active maintenance windows to extend, cancel, or modify them as needed.

## Managing windows

1. Navigate to **Maintenance** in the sidebar
2. Find the active or scheduled window
3. Click to view details
4. Use **Edit** to modify or **Cancel** to remove

## Extending maintenance

If maintenance takes longer than expected, edit the window to extend the end time. This prevents alerts from firing while work continues.
