## Overview

PageSpeed monitors report four main scores from Google Lighthouse audits. Each score ranges from 0-100, with higher scores indicating better performance.

## Score categories

- **Performance**: Measures loading speed and interactivity
- **Accessibility**: Evaluates usability for people with disabilities
- **Best practices**: Checks for modern web development standards
- **SEO**: Assesses search engine optimization basics

## Score ranges

| Score | Rating | Indicator |
| --- | --- | --- |
| 90-100 | Good | Green |
| 50-89 | Needs improvement | Orange |
| 0-49 | Poor | Red |

<Callout type="info">
Focus on Performance first, as it directly impacts user experience and SEO rankings.
</Callout>
