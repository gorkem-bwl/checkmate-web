## Overview

Performance trends show how your PageSpeed scores change over time. Use them to track the impact of optimizations and catch regressions.

## Viewing trends

1. Navigate to **PageSpeed** in the sidebar
2. Click on a monitor to view details
3. Select the time range (7 days, 30 days, etc.)
4. View the trend charts for each metric

## Interpreting trends

- **Upward trends**: Scores improving—your optimizations are working
- **Downward trends**: Scores declining—investigate recent changes
- **Flat trends**: Stable performance—no significant changes
- **Spikes**: Temporary issues—check for one-time events

<Callout type="info">
Compare trends before and after deployments to measure the impact of code changes on performance.
</Callout>
